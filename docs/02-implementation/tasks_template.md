# 実装タスクリスト

このファイルが実装フェーズの唯一の正の状態。会話がリセットされても、
ここを見れば同じ場所から再開できる。1タスク=1コミット粒度=1回の `task-worker` 呼び出し。

## 前提

| 項目 | 内容 |
|---|---|
| IRR判定 | TODO（GO / CONDITIONAL GO） |
| IRR承認日 | YYYY-MM-DD |
| CONDITIONAL GO条件（あれば） | TODO（条件IDと期限） |

## タスク

依存順に並べる。各タスクに設計ID・受け入れ条件IDを必ず付ける。
`CONDITIONAL GO` の未解決条件に依存するタスクは条件IDを明記し後方に置く。

- [ ] T-001: TODO（設計ID: DD-001 / 受け入れ条件: AC-001, AC-002）
- [ ] T-002: TODO（設計ID: DD-001 / 受け入れ条件: AC-003 / 依存: T-001）
